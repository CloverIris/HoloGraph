## [1.0.0] - 2026-04-10

### 初版发布 🎉

首个可用的 HoloGraph 桌面应用版本，包含完整的知识图谱核心功能。

### Added

#### 核心功能
- **知识节点管理** - 创建、编辑、删除知识节点
  - 支持 5 种节点类型：概念(concept)、笔记(note)、任务(task)、引用(reference)、标签(tag)
  - Markdown 内容编辑
  - 自定义颜色和标签系统
  - 自动时间戳记录

- **知识关联管理** - 建立节点间的连接关系
  - 可视化拖拽创建连接
  - 4 种关系类型：相关(related)、依赖(depends)、细化(refines)、序列(sequence)
  - 连接权重设置

- **可视化图谱** - 基于 SVG 的无限画布
  - 鼠标滚轮缩放 (10% - 500%)
  - 拖拽平移画布
  - 节点选择和多选
  - 适应视图功能

- **智能布局** - 自动排列算法
  - D3 力导向布局
  - 节点排斥力模拟
  - 连接边吸引力
  - 实时布局动画

- **路径规划** - Dijkstra 最短路径算法
  - 计算两点间最短知识路径
  - 路径高亮显示

- **搜索功能** - 全局知识检索
  - 实时搜索过滤
  - 按标签、内容、名称搜索
  - 快捷键 Ctrl+F

#### 数据管理
- **本地持久化** - 基于 localStorage
  - 自动保存（每 30 秒）
  - JSON 导入/导出
  - 数据版本兼容

#### 用户界面
- **Fluent Design** 设计系统
  - 深色主题
  - 亚克力背景效果
  - 流畅动画过渡
- **三栏布局**
  - 左侧：文件导航
  - 中央：图谱画布
  - 右侧：属性编辑
- **键盘快捷键**
  - 完整的快捷键支持
  - 快捷键提示

#### 桌面应用
- **Electron 封装**
  - Windows 原生应用
  - 便携版 (.exe)
  - 安装程序 (.exe)

### Technical Details

#### 技术栈
- React 18 + TypeScript 5.6
- Vite 5 构建工具
- Electron 33 桌面框架
- Zustand 状态管理
- @xyflow/react 可视化引擎
- D3-force 布局算法

#### 项目结构
```
src/
├── components/     # 12 个 React 组件
├── stores/         # 2 个 Zustand store
├── services/       # 存储服务
├── hooks/          # 自定义 Hooks
├── utils/          # 工具函数
└── styles/         # CSS 样式
```

### Known Issues
- 首次启动时可能有短暂白屏（正在加载资源）
- 大型图谱（500+ 节点）可能需要优化

## 版本说明

### 版本号格式
`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 修改
- **MINOR**: 向下兼容的功能新增
- **PATCH**: 向下兼容的问题修复

- 🎉 重大版本发布
- ✨ 新功能

- ⚡ 性能优化
